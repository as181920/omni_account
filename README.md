## Usage

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'omni_account', git: "git@github.com:as181920/omni_account.git", branch: "master"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install omni_account
```
